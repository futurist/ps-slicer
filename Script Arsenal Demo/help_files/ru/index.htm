Купить полную версию

"Script Arsenal" представляет собой набор сценариев для решения самых разнообразных задач в Photoshop.

Чтобы установить и подготовить его к работе:

1. Разархивируйте содержимое дистрибутива "script-arsenal-demo.zip" на жёсткий диск компьютера.

2. В главном меню Photoshop выберите "Файл > Сценарии > Обзор", найдите на своём компьютере папку "Script Arsenal Demo" и выберите в ней файл "scriptArsenalDemo.jsx".

3. В открывшемся диалоговом окне нажмите кнопку "Options" (рис. 1).

4. В меню "Language" выберите язык интерфейса диалогов (рис. 2).

5. Убедитесь, что установлен флажок "Add Script Arsenal to the main menu" (рис. 3).

6. Нажмите "ОК" для подтверждения настроек.

7. Перезапустите Photoshop и найдите "Script Arsenal" в меню программы "Файл > Сценарии > Script Arsenal" (File > Scripts > Script Arsenal).

Для более удобного доступа к этому сценарию рекомендуется также назначить ему клавиатурное сокращение и/или добавить его на панель Операций.
