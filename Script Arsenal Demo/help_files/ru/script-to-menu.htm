Купить полную версию

Чтобы ускорить доступ к часто используемым сценариям, их можно добавить в главное меню Photoshop .

Для этого откройте диалог "Script Arsenal", выделите нужный сценарий из списка и нажмите кнопку .
Новый скрипт появится в меню "Файл > Сценарии > ..." после перезапуска Photoshop.

Прим. Поскольку многие скрипты используют вспомогательные файлы из папки "Script Arsenal",
добавлять их в меню простым копированием в папку "Presets/Scripts" не рекомендуется.

Для ещё большего удобства добавленным скриптам можно назначить "горячие клавиши" (Редактирование >Клавиатурные сокращения). В примере, приведённом на скриншоте, диалог "Script Arsenal" открывается нажатием F4, а клавиша F5 служит для перехода к следующему файлу.

Прим. В Photoshop CS3 и CS4 вызов сценариев с помощью клавиатурных сокращений может работать некорректно. В этом случае, часто используемые скрипты лучше запускать с панели Операций (подробнее).
