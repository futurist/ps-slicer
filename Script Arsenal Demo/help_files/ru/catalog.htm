Купить полную версию

Это диалоговое окно предоставляет удобный доступ к сценариям, находящимся в папке каталога "Script Arsenal".

Полная версия каталога разбита на категории, переход между которыми осуществляется с помощью выпадающего меню. В демо-версии категории не поддерживаются.

Для воспроизведения сценария выберите его в списке и нажмите кнопку воспроизведения (скриншот) или дважды щёлкните строку скрипта.

Для получения краткой информации о выбранном сценарии выделите его в списке и нажмите кнопку "Info". (Если файл с информацией о скрипте отсутствует, кнопка не активируется.)

Если в глобальных настройках вы выбрали не английский вариант интерфейса, список отображает локализованные названия сценариев. Чтобы увидеть англоязычные названия, снимите галочку "Локализовать" (Localize) в нижней части диалога.

Надо также иметь в виду, что по умолчанию, выполнение каждого сценария отмечается на панели Истории одной строкой. Если вы хотите иметь возможность просматривать и отменять отдельные шаги, сделанные скриптом, снимите флажок "Свернуть историю" (Suspend History).
