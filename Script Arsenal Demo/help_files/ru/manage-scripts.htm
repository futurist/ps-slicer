Купить полную версию

Рассмотрим способы управления скриптами в диалоговом окне "Script Arsenal".

Чтобы добавить в каталог новый скрипт, нажмите кнопку "Add" и найдите на компьютере нужный JS или JSX файл.

Кнопка позволяет добавить выделенный сценарий в меню "Файл > Сценарии > ..." (подробнее).

Для переименования скрипта воспользуйтесь кнопкой "Rename".

Если понадобится изменить иконку скрипта:
- Подготовьте изображение размером 22 x 22 пикселя.
- Откройте диалог "Script Arsenal" (не закрывая созданное изображение!).
- Выделите в списке нужный сценарий.
- Нажмите кнопку "Make Thumbnail" (рис. 1).

Для удаления скрипта из каталога служит кнопка "Delete".

Кроме того, управлять сценариями можно и с помощью любого файлового менеджера. Каталог сценариев хранится в директории "... > Script Arsenal Demo > Scripts" (рис. 2).

Новые скрипты добавляются в каталог копированием JS или JSX файлов в папку "Scripts".

Для хранения информации о пользовательских настройках каждый скрипт использует собственный INI файл, находящийся в папке "... > Script Arsenal > Required > ini". В некоторых случаях, ошибка или повреждение такого файла может привести к неработоспособности сценария. Проблема решается удалением соответствующего INI файла.

Например, если перестал открываться диалог "Confetti", зайдите в папку "ini" и удалите файл "Confetti.ini".

Удалить INI файлы всех сценариев сразу, можно также из диалога глобальных настроек:

  1. Нажмите кнопку "Настройки" в нижней части диалога "Script Arsenal".
  2. В открывшемся диалоговом окне щёлкните кнопку общего сброса настроек.
