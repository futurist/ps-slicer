Script Arsenal ist ein Skriptenkatalog* für die Lösung verschiedenster Aufgaben in Photoshop.

So richten Sie den Katalog ein:

1. Entpacken Sie die Datei „script-arsenal-demo.zip“ auf der Festplatte Ihres Computers. Sie erhalten den Ordner „Script Arsenal Demo“.

2. Gehen Sie im Photoshop-Hauptmenü auf „Datei > Skripten > Durchsuchen“ und suchen Sie den Ordner „Script Arsenal“ auf Ihrer Festplatte. Starten Sie in dem Ordner die Datei „scriptArsenalDemo.jsx“, indem Sie sie anklicken.

3. Im Dialogfenster, das sich öffnet, klicken Sie auf „Optionen“ (Abb. 1).

4. Im Menü „Language“ wählen Sie die Dialogsprache aus (Abb. 2).

5. Überzeugen Sie sich, dass das Häkchen „Add Script Arsenal to the main menu“ gesetzt ist (Abb. 3).

6. Drücken Sie „OK“ um die Einstellungen zu übernehmen.

7. Nach dem Neustart von Photoshop finden Sie Script Arsenal im Hauptmenü des Programms unter „Datei > Skripten > Script Arsenal Demo“.

Zur Vereinfachung empfiehlt es sich, dem Skript „Script Arsenal“ eine Tastenkombination zuzuweisen und/oder es in das Aktionen-Bedienfeld einzufügen.

*Anmerkung: Zu dem Katalog gehört ein Ordner „Scripts Arsenal Demo“ auf der Festplatte des Computers. Der Dialog „Scripts Arsenal Demo“ dient als Schnittstelle für den Anwender.
