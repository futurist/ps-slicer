Für einen schnelleren Zugriff können Sie Skripten, die Sie oft benötigen, ins Hauptmenü von Photoshop einfügen.

Dazu öffnen Sie den Dialog „Script Arsenal“, wählen Sie das benötigte Skript aus der Liste aus und drücken Sie auf „Skript ins Menü einfügen“. Nach dem Neustart von Photoshop erscheint das neue Skript im Menü „Datei > Skripten > ...“.

Anmerkung: Da viele Skripten Hilfsdateien aus dem Ordner „Script Arsenal“ verwenden, empfiehlt es sich nicht, sie durch einfaches Kopieren in den Ordner „Presets/Scripts“ einzufügen.

Zur weiteren Arbeitserleichterung können Sie den Skripten Tastenkürzel zuweisen. Im Beispiel auf dem Screenshot öffnet sich der Dialog „Script Arsenal“ mit der Taste F4. Mit F5 kommt man zur nächsten Datei.

Anmerkung: In Photoshop CS3 und CS4 kann es vorkommen, dass der Aufruf von Skripten über Tastenkürzel nicht korrekt funktioniert. In diesem Fall ist es besser, wenn Sie die häufig benötigten Skripten über das Aktionen-Bedienfeld starten (mehr...).
