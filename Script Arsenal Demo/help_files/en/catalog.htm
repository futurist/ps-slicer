Get full version

This dialog window is intended for easy access to the scripts in the "Script Arsenal" catalog folder.

The full version of the catalog is divided into categories that can be selected in the drop-down menu. The demo version doesn't support categories.

To execute a script, you select it in the script panel and click the "Run" button (screenshot). Or just double-click the script name in the list.

To read a short description of a script select it in the list and click the "Info" button. (In case the script's info is unavailable, the button is not visible.)

In case, in the global options, you have selected a non-English language, the dialog shows the localized script names. To display the real file names instead, uncheck the "Localize" gadget at the dialog bottom.

Note also that, in the History panel, executing a script is recorded as a single history state. To enable undoing individual steps taken by each script, uncheck the "Suspend History" option.
