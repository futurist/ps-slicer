Get full version

To facilitate access to the most-often used scripts add those to the Photoshop menu.

To accomplish this, open the "Script Arsenal" dialog, select the appropriate script in the list, and click the button.
Once Photoshop is restarted, the new script appears in the "File > Scripts > ..." menu.

NB Many scripts coming to you with this set require auxiliary files from the "Script Arsenal" folder.
So, if you just copy such a script into the "Presets/Scripts" folder, it may not work.

For even faster access to the most used scripts, you may want to assign "hot keys" (Edit > Keyboard Shortcuts).
In the example shown on the screenshot, the "Script Arsenal" can be opened with F4 key press;
the F5 key allows to jump to the next file.

NB In Photoshop CS3 and CS4, launching scripts with keyboard shortcuts may not work properly.
In this case, consider running the scripts from the Actions panel (more about).
